## 下拉菜单
![dd](./img/bootstrap_droplist_1.jpg)

**注意事项**
在使用Bootstrap框架的下拉菜单时，必须调用Bootstrap框架提供的bootstrap.js文件。

**有关的class**
1.dropdown
2.dropdown-toggle
3.dropdown-menu

**有关的属性**
1.data-toggle

**实现原理**
就是通过控制`display:none` 以及`display:block`这两个属性来达到想要的效果的。
